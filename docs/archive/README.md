# Archive

This folder keeps completed, superseded, and historical planning material out of the active docs path.

## Completed

`completed/` contains implementation plans or phase plans whose shipped behavior now lives in code and stable guides.

## Historical

`historical/` contains older roadmaps or snapshots that still explain context, but should not be treated as current execution plans.

Root-level files in this archive predate the current docs split and are kept as-is unless a future cleanup pass needs finer grouping.
