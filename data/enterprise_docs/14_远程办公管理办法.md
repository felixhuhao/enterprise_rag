# 远程办公管理办法

**星辰科技 人力资源部**
**文档编号：** HR-REMOTE-2026-014
**生效日期：** 2026年3月1日
**版本：** V1.2
**适用范围：** 全体正式员工

---

## 一、总则

为适应现代办公方式，提升员工工作灵活性，平衡工作与生活，特制定本办法。远程办公是公司弹性工作制度的重要组成部分，旨在以结果为导向，在确保业务连续性的前提下，给予员工更大的工作自主权。

## 二、适用范围

| 项目 | 说明 |
|------|------|
| 适用对象 | 全体正式员工（已通过试用期转正） |
| 不适用对象 | 试用期员工、实习生、外包人员 |
| 生效条件 | 入职满6个月且转正评估通过 |

**注意：** 试用期员工如因特殊原因需远程办公，须经部门总监和HR总监双重审批，单次不超过2天。

## 三、申请与审批流程

### 3.1 申请流程

1. 员工提前至少1个工作日，在OA系统"远程办公申请"模块提交申请
2. 填写申请内容：远程日期、工作计划、紧急联系方式
3. 直属经理在申请当日完成审批（逾期未审批视为同意）

### 3.2 审批规则

| 远程天数 | 审批权限 |
|----------|----------|
| 单次1-2天 | 直属经理审批 |
| 单次3天及以上 | 直属经理 + 部门总监审批 |
| 全月累计超过5天 | 需额外报HR备案 |

### 3.3 频次限制

每位员工每月最多可申请**8天**远程办公。跨月不计入当月额度。法定节假日、公司统一放假期间不计入远程天数。

> **特别说明：** 本办法规定的远程办公天数上限（8天/月）与《员工手册》（文档编号：HR-MANUAL-2026-001）中考勤制度章节提及的弹性工作天数（5天/月）存在差异。经2026年2月管理层会议决议，自2026年3月1日起，远程办公天数上限调整为8天/月，《员工手册》相关条款将适时更新。在更新之前，以本办法为准。

## 四、设备与环境要求

### 4.1 设备要求

- 必须使用公司配发的笔记本电脑或台式机
- 禁止使用个人电脑处理公司业务数据
- 设备需安装公司统一的终端安全软件（包含防病毒、DLP数据防泄漏模块）
- 如需外接显示器等辅助设备，可向行政部申请借用

### 4.2 工作环境要求

| 要求项 | 具体标准 |
|--------|----------|
| 网络带宽 | 不低于20Mbps下行 / 5Mbps上行 |
| 工作空间 | 安静、独立的办公空间，确保通话质量 |
| 安全要求 | 禁止在公共场所（咖啡厅、图书馆等）处理机密及以上级别信息 |
| 电力保障 | 确保远程期间设备电力供应稳定 |

关于信息安全的具体要求，请严格遵守《信息安全管理制度》（文档编号：SEC-2026-003）中关于远程办公的安全规范。

## 五、网络接入要求

远程办公期间，访问公司内部系统必须通过VPN接入：

| 系统 | 访问方式 | VPN要求 |
|------|----------|---------|
| 邮件系统 | VPN + Web/客户端 | 必须开启VPN |
| 内部Wiki | VPN | 必须开启VPN |
| GitLab代码仓库 | VPN + SSH Key | 必须开启VPN |
| 飞书 | 直连（无需VPN） | 加密传输 |
| OA审批系统 | VPN | 必须开启VPN |

VPN使用规范详见《信息安全管理制度》（文档编号：SEC-2026-003）中的远程接入管理章节。禁止使用公共WiFi连接VPN，如必须使用需开启手机热点。

## 六、考勤与工作纪律

### 6.1 考勤要求

- 远程办公日仍需在飞书"打卡"功能中完成签到
- 签到时间：**9:00前**完成签到，18:00后签退
- 忘记签到需当日内在OA系统补签，每月补签不超过3次

### 6.2 工作纪律

| 要求 | 具体规定 |
|------|----------|
| 在线状态 | 工作时间内保持飞书在线状态 |
| 消息响应 | 工作消息30分钟内回复 |
| 会议响应 | 收到会议邀请后10分钟内确认参加或说明原因 |
| 电话响应 | 工作来电需及时接听或5分钟内回拨 |

## 七、绩效考核

远程办公不影响绩效考核，考核标准与在岗办公一致，以工作产出和结果为导向：

- **目标达成率：** 远程办公期间的OKR完成情况纳入季度考核
- **交付质量：** 代码质量、文档质量、项目进度不受办公方式影响
- **协作评价：** 团队成员对远程协作效率的评价纳入360度反馈

部门经理应关注远程办公员工的工作状态，必要时可要求员工返回办公室工作。

## 八、违规处理

| 违规行为 | 处理方式 |
|----------|----------|
| 未申请擅自远程办公 | 按旷工处理，扣减当日薪资 |
| 申请信息不实（如实际不在申报地点） | 警告，一年内累计3次取消远程资格 |
| 在公共场所处理机密信息 | 书面警告，情节严重按泄密事件处理 |
| 使用未授权设备 | 书面警告，设备检查 |
| 多次未按时签到 | 超过3次补签/月，暂停远程权限1个月 |

连续3个月远程办公天数超过8天的员工，直属经理需评估其是否适合继续远程办公，并向HR提交书面评估报告。

## 九、附则

- 本办法由人力资源部负责解释
- 本办法自2026年3月1日起施行，原《弹性工作管理细则》同时废止
- 本办法与《员工手册》不一致之处，以本办法为准
- 管理层保留根据业务需要调整远程办公政策的权利
